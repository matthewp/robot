# robot-hooks

Hooks utilities for use with [Robot](https://thisrobot.life/). Use this to create a `useMachine` hook that works with any hook-supporting view library.

## Installation

Via npm:

```shell
npm install robot-hooks --save
```

Or Yarn:

```shell
yarn add robot-hooks
```
